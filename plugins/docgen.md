# vuepress-plugin-docgen

> Vuepress plugin for creating a documentation site of your Vue components

## fast installation

  `yarn add -D vuepress-plugin-docgen`


vuepress config file

```jsx
plugins: [
  [ 'docgen',
    {
      componentsDir: path.join(__dirname, '../../components')
    }
  ]
]
```
