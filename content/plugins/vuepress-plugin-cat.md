# vuepress-plugin-cat

## How to use

1. install `npm i vuepress-plugin-cat -S`

2. add in `config.js`

```js
module.exports = {
  plugins: [
    "vuepress-plugin-cat",
  ]
}
```

## Tip

- This plugin supports `vuepress 1.x`
- This plugin will randomly change the white cat or black cat when router path change.
