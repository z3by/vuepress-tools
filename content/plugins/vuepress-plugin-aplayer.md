## Install

```npm
yarn add -D vuepress-plugin-aplayer
# OR npm install -D vuepress-plugin-aplayer
```

## Use

1). `config.js`

```javascript
module.exports = {
  plugins: ['aplayer']
}
```

2). `*.md`, [click here](https://aplayer.js.org/) for more configuration of aplayer, .

```markdown
<a-player 
    :options="{
        ...
    }"
/>
```
