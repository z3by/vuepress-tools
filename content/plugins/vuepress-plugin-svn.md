Use it the same way as [@vuepress/plugin-git](https://vuepress.github.io/reference/plugin/git.html)

## 安装

```sh
npm install vuepress-plugin-svn -D
```

## 使用

修改配置文件 `.vuepress/config.js`

```js
module.exports = {
  plugins: ["svn"],
};
```
